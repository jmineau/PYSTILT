## Installation

### From PyPI

```bash
pip install stilt
```

Or with [uv](https://github.com/astral-sh/uv):

```bash
uv pip install stilt
```

### From Source

```bash
git clone https://github.com/jmineau/PYSTILT.git
cd PYSTILT
pip install -e .
```

Or with uv:

```bash
git clone https://github.com/jmineau/PYSTILT.git
cd PYSTILT
uv pip install -e .
```

### For Development

With pip:

```bash
git clone https://github.com/jmineau/PYSTILT.git
cd PYSTILT
pip install -e ".[dev]"
pre-commit install
```

Or with uv (recommended for faster installation):

```bash
git clone https://github.com/jmineau/PYSTILT.git
cd PYSTILT
uv pip install -e ".[dev]"
pre-commit install
```
