Tutorials
=========

These tutorials stay focused on the parts of PYSTILT that are currently the
most solid:

- local trajectory and footprint runs
- Slurm execution
- footprint-driven flux calculations

Observation-heavy and Kubernetes service tutorials are intentionally deferred
until those alpha APIs settle further.
